# 🚀 Ultimate DSA Guide & Repository

Welcome to my comprehensive **Data Structures and Algorithms (DSA)** repository! This guide is a collection of my personal notes, code implementations, premium cheat sheets, and practical projects. I built this to not only track my own learning journey but to serve as a high-quality reference guide for anyone diving into DSA.

## 📂 Repository Structure

### 1️⃣ [Notes & Cheatsheets](./Notes_and_Cheatsheets/)
A curated collection of PDFs, diagrams, and interview guides.
- **Time Complexity Guides:** Deep dives into Big-O notation.
- **Interview Prep:** MAANG preparation approaches and standard interview Q&A.
- **Cheat Sheets:** Quick-reference diagrams and concept summaries.

### 2️⃣ [Data Structures Code](./Data_Structures_Code/)
Clean, well-commented C++ implementations of fundamental data structures.
- **[Linked Lists](./Data_Structures_Code/Linked_Lists/):** Singly, Doubly, and Circular linked lists.
- **[Stacks & Queues](./Data_Structures_Code/Stacks_and_Queues/):** Array and Linked-List based stacks, linear queues, circular queues, and double-ended queues.
- **[Trees & Heaps](./Data_Structures_Code/Trees_and_Heaps/):** Binary Search Trees (BST), Max Heaps, and Tree traversals.

### 3️⃣ [Algorithms](./Algorithms/)
Implementations of essential algorithmic techniques.
- **Sorting:** Selection sort, insertion sort, etc.

### 4️⃣ [Practical Projects](./Projects/)
Where theory meets practice! Small CLI applications built to demonstrate data structures in action.
- **[Event Registration System](./Projects/Event_Registration_System/):** Manages participants using a singly linked list.
- **[Plate Game](./Projects/Plate_Game/):** A stack-based logic game.

### 5️⃣ [Practice Problems](./Practice_Problems/)
PDF lists of top practice problems, including 100+ LeetCode essentials and topic-wise breakdown sheets.

## 🌟 How to Use This Guide
1. **Understand the Theory:** Start by reading through the PDFs in `Notes_and_Cheatsheets`.
2. **Study the Code:** Check out the raw implementations in `Data_Structures_Code`. Run them locally to see how they work.
3. **Build Projects:** Look at the `Projects/` folder to see how these concepts are used to build real applications.
4. **Practice:** Solve the problems provided in `Practice_Problems`.

## 🛠️ Tech Stack
- **Language:** C++
- **Focus:** Object-Oriented Programming, Memory Management (Pointers), Data Structures, Algorithms.
